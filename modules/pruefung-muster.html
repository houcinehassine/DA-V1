Muster: Loops über Dicts

→ Ü2 Klausur
  • Nested Dict iterieren mit verschachtelten for-Loops
  • Kleinsten Wert finden ohne min() (manuell!)
  • while-Loop mit random.randint() – Zahlen raten
# Nested Dict (OTH Muster)
daten = {
    "Informatik": {"StudiAnz": 350, "Profs": 12},
    "BWL":        {"StudiAnz": 280, "Profs": 8}
}
for fach, info in daten.items():
    print(f"{fach}: {info['StudiAnz']} Studierende")

# Kleinsten Wert manuell finden
zahlen = [34, 12, 67, 5, 89, 23]
kleinster = zahlen[0]
for z in zahlen:
    if z < kleinster:
        kleinster = z
print(f"Kleinster: {kleinster}")

# Zahlen raten (while + break)
import random
zahl = random.randint(1, 100)
versuche = 0
while True:
    tipp = int(input("Tipp: "))
    versuche += 1
    if tipp == zahl:
        print(f"Richtig nach {versuche} Versuchen!")
        break
    elif tipp < zahl:
        print("Zu klein")
    else:
        print("Zu groß")

Muster: Funktionen mit Statistik

→ Ü3 Klausur
  • Deskriptive Statistik-Funktion: min, max, mean, std
  • Error Handling für ungültige Eingaben
  • Funktion nimmt Dict oder Liste, gibt Dict zurück
import math

def descriptive_stats(data: list) -> dict:
    if not data:
        raise ValueError("Leere Liste!")
    n    = len(data)
    mean = sum(data) / n
    variance = sum((x - mean)**2 for x in data) / n
    std  = math.sqrt(variance)
    return {
        "n":    n,
        "min":  min(data),
        "max":  max(data),
        "mean": round(mean, 2),
        "std":  round(std, 2)
    }

werte = [12, 45, 23, 67, 34, 89, 11]
ergebnis = descriptive_stats(werte)
for k, v in ergebnis.items():
    print(f"{k:<5}: {v}")

Muster: Neural Network Init & Forward

→ Ü5 Klausur
  • Gewichtsmatrizen mit np.random.rand(n_in, n_hid)
  • Bias-Vektoren mit np.zeros(n_hid)
  • Feed-Forward: z = np.dot(X, W) + b
  • Aktivierung: z.B. ReLU, Sigmoid
import numpy as np

# Netz-Architektur
n_inputs  = 4
n_hidden  = 6
n_outputs = 3

# Initialisierung
W1 = np.random.rand(n_inputs,  n_hidden)  * 0.01
b1 = np.zeros(n_hidden)
W2 = np.random.rand(n_hidden,  n_outputs) * 0.01
b2 = np.zeros(n_outputs)

# Feed-Forward
X = np.random.rand(10, n_inputs)  # 10 Datenpunkte
z1 = np.dot(X,  W1) + b1          # (10, 6)
a1 = np.maximum(0, z1)            # ReLU
z2 = np.dot(a1, W2) + b2          # (10, 3)

print("Output shape:", z2.shape)  # (10, 3)
