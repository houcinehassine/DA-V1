🔬 Data Science Project: Loss in Coupling

Projekt 1

Loss in Coupling – Analyse von Konnektivitätsmatrizen

numpy · correlation · matrix · EEG · neuroscience

In diesem Projekt analysierst du EEG-Konnektivitätsdaten. Jeder Proband hat mehrere "Epochen" – kurze Zeitabschnitte – für zwei Bedingungen: Ruhe und Aufgabe. Für jede Epoche liegt eine Korrelationsmatrix (Konnektivitätsmatrix) zwischen EEG-Kanälen vor.

Ziel: Finde heraus, ob sich die mittlere Konnektivität zwischen den Bedingungen unterscheidet.

Schritt 1

Daten laden & Struktur verstehen

numpy · shape · ndarray

Lade die Daten und verschaffe dir einen Überblick: Wie viele Probanden, Epochen, Kanäle? Was ist die Shape der Arrays?

AUFGABE
import numpy as np

np.random.seed(42)
n_subjects = 20
n_epochs   = 30
n_channels = 32

# Simuliere Konnektivitätsmatrizen (symmetrisch, Diagonale = 1)
def make_conn_matrices(n_subj, n_ep, n_ch, base_coupling):
    data = np.zeros((n_subj, n_ep, n_ch, n_ch))
    for s in range(n_subj):
        for e in range(n_ep):
            M = np.random.randn(n_ch, n_ch) * 0.1 + base_coupling
            M = (M + M.T) / 2
            np.fill_diagonal(M, 1.0)
            data[s, e] = M
    return data

rest_data = make_conn_matrices(n_subjects, n_epochs, n_channels, base_coupling=0.3)
task_data = make_conn_matrices(n_subjects, n_epochs, n_channels, base_coupling=0.5)

print("Rest shape:", rest_data.shape)
print("Task shape:", task_data.shape)
✅ Musterlösung
LÖSUNG
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.RandomState(42)
n_channels = 5  # Anzahl der Sensoren/Kanäle

# Simuliere 4 Wochen Messdaten als Korrelationsmatrizen
# Wochen 1-3: normale Kopplung (niedrige Korrelation)
# Woche 4: erhöhte Kopplung (möglicher Schaden)
matrices = []
for week in range(4):
    if week < 3:
        # Normalzustand: schwache Korrelationen zwischen 0.1 und 0.4
        corr = rng.uniform(0.1, 0.4, (n_channels, n_channels))
    else:
        # Schadenszustand: starke Korrelationen zwischen 0.5 und 0.9
        corr = rng.uniform(0.5, 0.9, (n_channels, n_channels))

    # Symmetrisch machen: (M + M^T) / 2 → echte Korrelationsmatrix
    corr = (corr + corr.T) / 2
    # Diagonale auf 1 setzen: ein Kanal korreliert 100% mit sich selbst
    np.fill_diagonal(corr, 1.0)
    matrices.append(corr)

# Mittlere Kopplung pro Woche: nur oberes Dreieck (ohne Diagonale) betrachten
# np.triu_indices gibt die Indizes der oberen Dreiecksmatrix zurück
mean_corr = [np.mean(m[np.triu_indices(n_channels, k=1)]) for m in matrices]

print("Mittlere Kopplung pro Woche:", [f"{v:.3f}" for v in mean_corr])
print(f"→ Erhöhung in Woche 4: {mean_corr[3] / mean_corr[0]:.1f}x")

Schritt 3

Obere Dreiecksmatrix extrahieren

numpy · triu_indices · flatten · upper triangle

Extrahiere nur die Werte aus dem oberen Dreieck (ohne Diagonale) jeder gemittelten Matrix. Das vermeidet Redundanz (die Matrix ist symmetrisch).

AUFGABE
# Indices des oberen Dreiecks (k=1 = ohne Diagonale)
idx = np.triu_indices(n_channels, k=1)

# Für jeden Probanden: extrahiere oberes Dreieck als Vektor
rest_vectors = np.array([rest_mean[s][idx] for s in range(n_subjects)])
task_vectors = np.array([task_mean[s][idx] for s in range(n_subjects)])

print("rest_vectors shape:", rest_vectors.shape)
# Erwartete Shape: (20, 496) — 32*31/2 = 496 unique pairs
Schritt 4

Mittlere Konnektivität pro Proband

numpy · mean · comparison

Berechne für jeden Probanden die mittlere Konnektivität (Mittelwert aller Paar-Korrelationen) für Ruhe und Aufgabe. Vergleiche beide Bedingungen.

AUFGABE
# Mittlere Konnektivität je Proband (mean über alle Paare)
rest_conn = rest_vectors.mean(axis=1)   # Shape: (20,)

# Differenz pro Proband
diff = task_conn - rest_conn

print(f"Rest: {rest_conn.mean():.3f} ± {rest_conn.std():.3f}")
print(f"Task: {task_conn.mean():.3f} ± {task_conn.std():.3f}")
print(f"Diff: {diff.mean():.3f} ± {diff.std():.3f}")
print(f"Alle Probanden zeigen Anstieg: {(diff > 0).all()}")
Schritt 5

Statistischer Test: t-Test

scipy · ttest_rel · p-value · effect size

Führe einen gepaarten t-Test durch, um zu prüfen ob der Unterschied zwischen Ruhe und Aufgabe statistisch signifikant ist. Berechne zusätzlich Cohen's d als Effektgröße.

AUFGABE
from scipy import stats

# Gepaarter t-Test (jeder Proband ist seine eigene Kontrolle)
t_stat, p_value = stats.ttest_rel(task_conn, rest_conn)

# Cohen's d = mean(diff) / std(diff)
cohens_d = diff.mean() / diff.std()

print(f"t = {t_stat:.3f}, p = {p_value:.4f}")
print(f"Cohen's d = {cohens_d:.3f}")
print(f"Signifikant (p < 0.05): {p_value < 0.05}")

Projekt 2

Tracking Connectivity States über die Zeit

numpy · clustering · k-means · time series · dynamic FC

Dynamische funktionelle Konnektivität (dynamic FC) untersucht, wie sich Gehirnnetzwerke über die Zeit ändern. Statt eine einzige mittlere Matrix zu berechnen, schauen wir, wie die Konnektivität von Epoche zu Epoche wechselt.

Ziel: Finde diskrete "Konnektivitätszustände" (connectivity states) mithilfe von K-Means-Clustering auf den Epochen-Konnektivitätsvektoren.

AUFGABE
import numpy as np
from sklearn.cluster import KMeans

np.random.seed(0)
n_epochs_total = 200
n_channels = 16

# Simuliere 3 verschiedene Konnektivitätszustände
state_templates = [
    np.random.rand(n_channels, n_channels) * 0.1 + 0.2,   # Zustand 1: niedrig
    np.random.rand(n_channels, n_channels) * 0.1 + 0.5,   # Zustand 2: mittel
    np.random.rand(n_channels, n_channels) * 0.1 + 0.8,   # Zustand 3: hoch
]

# Jede Epoche gehört zufällig zu einem der 3 Zustände
true_labels = np.random.choice(3, n_epochs_total)
epochs = np.array([
    state_templates[l] + np.random.randn(n_channels, n_channels) * 0.05
    for l in true_labels
])

# Extrahiere oberes Dreieck je Epoche
idx = np.triu_indices(n_channels, k=1)
epoch_vectors = np.array([epochs[e][idx] for e in range(n_epochs_total)])

# 1. K-Means mit k=3 auf epoch_vectors
kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
predicted_labels = kmeans.fit_predict(epoch_vectors)

# 2. Wie oft wechselt der Zustand?
n_switches = (np.diff(predicted_labels) != 0).sum()
print(f"Anzahl Zustandswechsel: {n_switches} von {n_epochs_total-1} möglichen")

LÖSUNG
import numpy as np, pandas as pd
import matplotlib.pyplot as plt, seaborn as sns

np.random.seed(42)
n = 100
group_a = np.random.randn(n,1)*np.ones((1,4)) + np.random.randn(n,4)*0.2
group_b = np.random.randn(n,1)*np.ones((1,3)) + np.random.randn(n,3)*0.2
ts = np.hstack([group_a, group_b])

# Korrelationsmatrix: jede Spalte = eine Zeitserie
# np.corrcoef erwartet (n_variables, n_samples) → transponieren
corr = np.corrcoef(ts.T)

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
sns.heatmap(corr, annot=True, fmt='.2f', ax=axes[0], cmap='coolwarm', vmin=-1, vmax=1)
axes[0].set_title('Korrelationsmatrix')

# Clustermap: gruppiert ähnliche Zeitserien automatisch
sns.clustermap(pd.DataFrame(corr), cmap='coolwarm', vmin=-1, vmax=1)
plt.suptitle('Clustermap – Gruppen sichtbar', y=1.02)
plt.tight_layout(); plt.show()
EXA 2

Fault Detection – Hypothesentest

scipy.stats.ttest_ind · std · histogram · distplot · p-value

Drei Motoren gleichen Typs: Motor 1 ist funktionsfähig. Schaden zeigt sich in der Standardabweichung. Überprüfe mittels t-Test (scipy.stats.ttest_ind), ob Motoren 2 und 3 signifikant abweichen.

  • Berechne Standardabweichung aller Zeitserien jedes Motors
  • Vergleiche Histogramme der Std-Werte
  • t-Test: H₀: gleiche Verteilung wie Motor 1
  • p < 0.05 → signifikant verschieden → Motor fehlerhaft
AUFGABE
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# engine_1 = np.load('./data/engine_1.npy')  # shape: (n_samples, n_time)
# Simulation: Motor 1 normal, Motor 2 und 3 mit erhöhter Std
np.random.seed(0)
engine_1 = np.random.randn(50, 200)         # normal
engine_2 = np.random.randn(50, 200) * 2.5   # erhöhte Std
engine_3 = np.random.randn(50, 200) * 0.4   # reduzierte Std

# 1. Berechne Std pro Zeitserie: np.std(engine, axis=1)
# 2. Plotte Histogramme aller 3 Motoren
# 3. t-Test: ttest_ind(std_engine_1, std_engine_2) und Motor 3
# 4. Interpretiere p-Wert
✅ Lösung
LÖSUNG
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

np.random.seed(0)
engine_1 = np.random.randn(50, 200)
engine_2 = np.random.randn(50, 200) * 2.5
engine_3 = np.random.randn(50, 200) * 0.4

std_1 = np.std(engine_1, axis=1)
std_2 = np.std(engine_2, axis=1)
std_3 = np.std(engine_3, axis=1)

plt.figure(figsize=(10, 4))
plt.hist(std_1, alpha=0.6, label='Motor 1 (OK)')
plt.hist(std_2, alpha=0.6, label='Motor 2')
plt.hist(std_3, alpha=0.6, label='Motor 3')
plt.legend(); plt.xlabel('Standardabweichung'); plt.title('Verteilung der Std pro Zeitserie')
plt.tight_layout(); plt.show()

t2, p2 = stats.ttest_ind(std_1, std_2)
t3, p3 = stats.ttest_ind(std_1, std_3)
print(f"Motor 2: t={t2:.2f}, p={p2:.4f} → {'FEHLERHAFT' if p2 < 0.05 else 'OK'}")
print(f"Motor 3: t={t3:.2f}, p={p3:.4f} → {'FEHLERHAFT' if p3 < 0.05 else 'OK'}")
