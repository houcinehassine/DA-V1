Data Understanding II: Paarweise & Multivariate Analysen

Beziehungen zwischen Variablen verstehen — Scatter Plots, Korrelation, Pair Plots, Hypothesentests und die Warnung von Anscombe's Quartet.

Inhalte
→ Scatter Plot → Joint Plots (hex, kde, reg) → Korrelationskoeffizient → Pair Plots → Korrelationsmatrix → t-Test & Hypothesentest → Anscombe's Quartet ⚠️

🔵 Scatter Plot

Ein Scatter Plot (Streudiagramm) zeigt die Beziehung zwischen zwei numerischen Variablen. Jeder Punkt = eine Beobachtung.

PYTHON
import plotly.express as px
import matplotlib.pyplot as plt

# Plotly (interaktiv)
fig = px.scatter(df, x='variable_1', y='variable_2',
                 opacity=0.75, title='Scatter Plot')
fig.show()

# Matplotlib (klassisch)
plt.scatter(df['variable_1'], df['variable_2'], alpha=0.5)
plt.xlabel('Variable 1')
plt.ylabel('Variable 2')
plt.show()
↗️
Positiv korreliert
x steigt → y steigt
↘️
Negativ korreliert
x steigt → y sinkt
🌐
Kein Zusammenhang
Punkte zufällig verteilt

📊 Joint Plots

Ein Joint Plot (Seaborn) kombiniert den Scatter Plot mit Randverteilungen — sehr praktisch für erste Datenexploration.

PYTHON
import seaborn as sns

# scatter: klassischer Scatter mit Histogrammen am Rand
sns.jointplot(data=df, x='var_1', y='var_2', kind='scatter')

# hex: Hexbin für dichte Datenpunkte (verhindert Overplotting)
sns.jointplot(data=df, x='var_1', y='var_2', kind='hex')

# kde: Kernel Density Estimation (Wahrscheinlichkeitsdichte)
sns.jointplot(data=df, x='var_1', y='var_2', kind='kde')

# reg: Mit Regressionsgerade + Konfidenzintervall
sns.jointplot(data=df, x='var_1', y='var_2', kind='reg', ci=95)
kind= Wann nutzen?
'scatter'Wenige Datenpunkte (< 1.000)
'hex'Viele Datenpunkte (Overplotting vermeiden)
'kde'Kontinuierliche Wahrscheinlichkeitsdichte zeigen
'reg'Linearen Zusammenhang modellieren und visualisieren

import numpy as np
from scipy.stats import linregress

# Korrelation zwischen zwei Variablen
r = df['var_1'].corr(df['var_2'])
print(f"r = {r:.3f}")

# Mit scipy: r + p-Wert

⚠️
Korrelation ≠ Kausalität!

Nur weil zwei Variablen korreliert sind, bedeutet das nicht, dass eine die andere verursacht. Beispiel: Eisverkauf und Ertrinkungsunfälle korrelieren beide mit Sommerhitze.

🔢 Pair Plots (Multivariate Analyse)

Ein Pair Plot zeigt alle paarweisen Scatter Plots zwischen mehreren Variablen gleichzeitig — ideal für einen ersten Überblick über einen Datensatz.

PYTHON
import seaborn as sns

# Einfacher Pair Plot
sns.pairplot(data=df)

# Mit Gruppenfärbung (hue)
sns.pairplot(data=df, hue='label')   # färbt nach Klasse/Gruppe
💡
Was zeigt die Diagonale?

Auf der Diagonale eines Pair Plots steht die Verteilung jeder einzelnen Variable (Histogramm oder KDE). Die restlichen Felder zeigen paarweise Scatter Plots.

🗺️ Korrelationsmatrix (Heatmap)

PYTHON
import matplotlib.pyplot as plt

# Korrelationsmatrix berechnen
corr_mat = df.corr()
print(corr_mat)

#             var_1    var_2    var_3
# var_1     1.000   -0.183    0.904
# var_2    -0.183    1.000   -0.453
# var_3     0.904   -0.453    1.000

# Als Heatmap visualisieren
plt.imshow(corr_mat, cmap='RdBu', vmin=-1, vmax=1)
plt.colorbar()
plt.xticks(range(len(corr_mat)), corr_mat.columns, rotation=45)
plt.yticks(range(len(corr_mat)), corr_mat.columns)
plt.title('Korrelationsmatrix')
plt.show()

🧪 Hypothesentest: t-Test

Mit dem t-Test prüfen wir, ob der Unterschied zwischen zwei Gruppen statistisch signifikant ist — oder ob er zufällig entstanden sein könnte.

PYTHON
from scipy import stats

data_a = [1, 2, 1, 2, 3, 1, 2]
data_b = [4, 5, 4, 6, 3, 2, 1]

result = stats.ttest_ind(data_a, data_b)
print(result)
# Ttest_indResult(statistic=-2.617, pvalue=0.022)
Ergebnis interpretieren
statistic
Der t-Wert: Je weiter von 0, desto größer der Unterschied zwischen den Gruppen
p-value
Wahrscheinlichkeit, dieses Ergebnis durch Zufall zu sehen. p < 0.05 → signifikant
✅
p < 0.05

Signifikant! Der Unterschied ist wahrscheinlich kein Zufall.

⚠️
p ≥ 0.05

Nicht signifikant — kein ausreichender Beweis für einen Unterschied.

⚠️ Anscombe's Quartet — Immer visualisieren!

Anscombe's Quartet zeigt 4 Datensätze, die identische statistische Kennzahlen haben (gleicher Mittelwert, gleiche Varianz, gleiche Korrelation) — aber völlig unterschiedliche Verteilungen.

🚨 Die 4 Datensätze haben alle:
• Mittelwert x ≈ 9.0
• Mittelwert y ≈ 7.5
• Varianz x ≈ 11.0
• Varianz y ≈ 4.1
• Korrelation r ≈ 0.816
• Regressionsgerade identisch
→ Aber im Plot sieht jeder Datensatz VÖLLIG anders aus!
PYTHON
import seaborn as sns

# Anscombe-Datensatz laden
df_anscombe = sns.load_dataset("anscombe")

# Alle 4 Gruppen visualisieren
sns.lmplot(x="x", y="y",
           col="dataset",     # je Datensatz ein Plot
           hue="dataset",
           data=df_anscombe,
           col_wrap=2, ci=None)
🔴
Die Lektion: Immer zuerst visualisieren!

Statistiken alleine reichen nicht — die Grafik zeigt, was die Zahlen verschweigen. Outlier, Kurvenformen, Clustern: Nur im Plot erkennbar.

📋 Teste dein Wissen — Data Understanding II
1. Was bedeutet r = −0.85?
Starker positiver Zusammenhang
Starker negativer Zusammenhang (x steigt → y sinkt)
Kein Zusammenhang
2. Was ist das Problem mit der Aussage "Eisverkauf verursacht Ertrinkungsunfälle, weil sie korrelieren"?
Die Korrelation ist falsch berechnet
Korrelation bedeutet nicht Kausalität — beide werden durch eine dritte Variable (Hitze) verursacht
Man müsste einen t-Test durchführen
3. Welches kind beim jointplot nutzt man bei sehr vielen Datenpunkten, um Overplotting zu vermeiden?
'scatter'
'hex'
'reg'
4. Was lehrt uns Anscombe's Quartet?
Statistiken wie Mittelwert und Korrelation reichen aus, um Daten zu verstehen
Man sollte Daten immer visualisieren — gleiche Statistiken können völlig verschiedene Verteilungen verbergen
Der t-Test ist die wichtigste statistische Methode
