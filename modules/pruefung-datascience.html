NumPy Arrays & Operationen

→ Modul 06
  • Elementweise: A * B ≠ Matrix-Multiplikation → dafür np.dot(A, B)
  • Boolean Indexing: arr[arr > 3] → Werte über 3
  • axis: axis=0 spaltenweise (über Zeilen), axis=1 zeilenweise (über Spalten)
  • Neural-Net-Muster: np.random.rand(n_in, n_hid) → Gewichtsmatrix
import numpy as np

# Arrays erstellen
a = np.array([1, 2, 3, 4, 5])
m = np.zeros((3, 4))            # 3×4 Nullen
r = np.random.rand(3, 3)        # gleichverteilt [0,1]
rn = np.random.randn(3, 3)      # normalverteilt

# Statistik
np.mean(a);  np.std(a);  np.var(a)
np.max(a);   np.min(a);  np.sum(a)
np.argmax(a)                     # Index des Maximums
np.std(m, axis=1)                # Zeilenweise

# Indexing
m[0, :]      # erste Zeile
m[:, 1]      # zweite Spalte
a[a > 3]     # Boolean → [4, 5]
a[(a > 1) & (a < 5)]  # kombiniert

# Reshaping
a.reshape(5, 1)   # Spaltenvektor
a.reshape(-1)     # → 1D flatten
np.vstack([a, a]) # vertikal stapeln (rows)
np.hstack([a, a]) # horizontal stapeln

# Matrix-Multiplikation
z = np.dot(X, W) + b   # Neurales Netz: Layer-Output

Pandas DataFrames

→ Modul 06
  • .describe(): count, mean, std, min, 25%, 50%, 75%, max
  • .groupby(): aggregiert nach Kategorien
  • .rolling(n): n-Perioden gleitendes Fenster – sehr klausurrelevant!
  • Filter: df[df["col"] > wert]
import pandas as pd

df = pd.read_csv("daten.csv")
df = pd.DataFrame({"a": [1,2,3], "b": [4,5,6]})

# Überblick
df.head(5)        # erste 5 Zeilen
df.info()         # Datentypen + NaN-Zählung
df.describe()     # stat. Kennzahlen
df.shape          # (n_rows, n_cols)

# Zugriff
df["spalte"]                  # Spalte → Series
df[["a", "b"]]                # mehrere Spalten
df[df["a"] > 1]              # Filter
df.iloc[0]                    # erste Zeile (Pos.)
df.loc[df["a"] > 1, "b"]     # Filter + Spalte

# Aggregation
df.groupby("kategorie")["wert"].mean()
df.groupby("kat").agg({"wert": ["mean","std"]})

# Rolling (7-Tage-Inzidenz / gleitend)
df["7d_sum"] = df["faelle"].rolling(7).sum()

Korrelation & Visualisierung

→ Modul 07
  • Korrelation: Wert −1 … +1. +1 = perfekt positiv, 0 = kein Zusammenhang, −1 = perfekt negativ
  • np.corrcoef(data.T): data hat Shape (n_samples, n_features) → .T transponieren!
  • sns.clustermap(): gruppiert ähnliche Features automatisch
  • annot=True: zeigt Werte in der Heatmap an
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt

# Korrelationsmatrix
corr = np.corrcoef(data.T)  # data: (n_samples, n_features)

corr = df.corr()

sns.heatmap(corr,

Hypothesentest (t-Test)

→ Modul 07
  • H₀ (Nullhypothese): kein Unterschied zwischen den Gruppen (µ₁ = µ₂)
  • H₁ (Alternativhypothese): Unterschied vorhanden (µ₁ ≠ µ₂)
  • p < 0.05: signifikant → H₀ ablehnen (Unterschied ist real)
  • p ≥ 0.05: nicht signifikant → H₀ beibehalten
from scipy import stats

# t-Test für 2 unabhängige Gruppen
t_stat, p_value = stats.ttest_ind(group1, group2)

print(f"t-Statistic: {t_stat:.4f}")
print(f"p-Value:     {p_value:.4f}")

if p_value < 0.05:
    print("✓ Signifikanter Unterschied (H0 ablehnen)")
else:
    print("✗ Kein signifikanter Unterschied")

# Gruppen aus DataFrame herausfiltern
group1 = df[df["label"] == "normal"]["wert"].values
group2 = df[df["label"] == "fehler"]["wert"].values

# Visualisierung der Gruppen
import matplotlib.pyplot as plt
plt.boxplot([group1, group2], labels=["Normal", "Fehler"])
plt.title(f"p = {p_value:.4f}")
plt.show()
