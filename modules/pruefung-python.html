🐍 Python Grundlagen Module 02–04

Variablen & Datentypen

→ Modul 02
  • Naming: snake_case, keine Zahlen am Anfang, keine Reserved Words
  • Typen: int, float, str, bool, list, dict, tuple, set
  • Falsy: 0, 0.0, "", [], {}, None → alle ergeben bool(x) == False
  • Konvertierung: int(), float(), str(), bool()
  • Operatoren: // ganzzahlig, % Rest, ** Potenz
  • f-String: f"Text {variable:.2f}" → 2 Dezimalstellen
alter = 25           # int
pi = 3.14159         # float
name = "Max"         # str
aktiv = True         # bool

# Typkonvertierung
int("42")    # → 42
float("3.14")# → 3.14
str(100)     # → "100"
bool(0)      # → False

# Arithmetik
10 // 3  # → 3   (ganzzahlig)
10 % 3   # → 1   (Rest / Modulo)
2 ** 3   # → 8   (Potenz)

# f-String
print(f"{name} ist {alter} Jahre alt, Pi ≈ {pi:.2f}")

Listen & Dictionaries

→ Modul 02
  • Indexing: lst[0] erstes, lst[-1] letztes, lst[-2] vorletztes
  • Slicing: lst[1:3], lst[::2] jedes 2., lst[::-1] umgekehrt
  • Mutable: list, dict, set – veränderbar. Immutable: str, tuple – unveränderbar
  • dict.get(key, default): kein KeyError wenn Key fehlt
lst = [10, 20, 30, 40, 50]
lst[0]      # → 10  lst[-1]  → 50
lst[1:3]    # → [20, 30]
lst[::2]    # → [10, 30, 50]
lst[::-1]   # → [50, 40, 30, 20, 10]

lst.append(60)      # hinten anhängen
lst.insert(0, 5)    # an Position 0
lst.remove(30)      # Wert 30 entfernen
lst.pop()           # letztes entfernen
len(lst)            # Länge

# Strings = immutable Liste
s = "Hallo"
s.upper()           # → "HALLO"
s.replace("l","L")  # → "HaLLo"
s.split("l")        # → ["Ha", "", "o"]
s.find("ll")        # → Index (oder -1)

# Dict
person = {"name": "Max", "alter": 25}
person["name"]              # → "Max"
person.get("job", "N/A")    # → "N/A" (kein KeyError!)
person["stadt"] = "München" # neuer Key
del person["alter"]         # Key löschen
for k, v in person.items(): # iterieren
    print(f"{k}: {v}")

Loops & Conditions

→ Modul 03
  • for wenn Anzahl bekannt (Liste, range, dict)
  • while wenn Bedingung geprüft wird – immer Abbruchbedingung!
  • List Comprehension: äquivalent zu for + append, kürzer
  • zip() parallele Listen, enumerate() Index + Wert
for i in range(1, 6):        # 1 2 3 4 5
    print(i)

for k, v in person.items():  # Dict iterieren
    print(f"{k} = {v}")

for a, b in zip(list1, list2):  # parallel
    print(a, b)

# while – immer Abbruchbedingung!
n = 0
while n < 5:
    n += 1

# break / continue
for i in range(10):
    if i == 3: continue    # 3 überspringen
    if i == 7: break       # bei 7 stoppen

# if / elif / else
if x > 10:
    print("groß")
elif x > 5:
    print("mittel")
else:
    print("klein")

# List Comprehension
quadrate = [x**2 for x in range(1, 6)]
gerade   = [x for x in range(20) if x % 2 == 0]

Funktionen & Error Handling

→ Modul 04
  • 4 Formen: ohne Ein/Ausgabe | mit Eingabe | mit Ausgabe | mit beidem
  • Default-Parameter: def f(x, y=0): – immer am Ende!
  • Mehrere Rückgabewerte als Tuple: return a, b → x, y = f()
  • try/except: ValueError, TypeError, ZeroDivisionError
  • Lambda: einzeilige anonyme Funktion
def berechne_bmi(gewicht, groesse, einheit="kg"):
    bmi = gewicht / groesse**2
    return round(bmi, 1)

# Mehrere Rückgabewerte
def stats(lst):
    return min(lst), max(lst), sum(lst)/len(lst)

lo, hi, avg = stats([1, 2, 3, 4, 5])

# Error Handling
def safe_convert(s):
    try:
        return float(s)
    except ValueError:
        return None

# Lambda + sorted
names = ["Bob", "Alice", "Charlie"]
sorted(names, key=lambda x: len(x))  # nach Länge

# map / filter
sq = list(map(lambda x: x**2, [1,2,3,4]))
ev = list(filter(lambda x: x%2==0, [1,2,3,4]))

# Domain ersetzen (Klausur-Muster!)
def replace_domain(email, old, new):
    if old not in email:
        raise ValueError(f"Domain '{old}' nicht gefunden")
    return email.replace(old, new)
