Loops & Conditions

Wie wiederholt man Aktionen automatisch? Wie trifft man Entscheidungen im Code? for-Loops, while-Loops, if/elif/else und List Comprehensions.

Inhalte dieses Moduls
→ for-Loop → range() & Indentation → Parallele Listen + Index → while-Loop → for vs. while → if / elif / else → Mehrere if-Statements → break & continue → List Comprehensions → Selbsttest

🔁 for-Loop

📐
Formale Definition

Abgeschlossene Loops (for-Loops) haben explizite, vordefinierte Iterationsvariablen, die sich mit jeder Iteration verändern. Sie bewegen sich durch das Iterable, um das der Loop definiert ist.

Iterable = ein Objekt, über das iteriert werden kann: Listen, Strings, range(), Dictionaries…
"An iterable is an object, which can be iterated over."

💡
Warum Loops? — Repetitive Aufgaben automatisieren

Looping ist eines der wichtigsten Konzepte des Programmierens — es ist der Weg um repetitive Aufgaben zu automatisieren. Stell dir vor: 100 Excel-Dateien nacheinander laden, 100-mal denselben Copy-Paste-Vorgang. Mit einem for-Loop geht das in wenigen Zeilen:

PYTHON
# Ohne Loop: 100× copy-paste
# print("Datei my_document1.xls wird geladen")
# print("Datei my_document2.xls wird geladen")
# ...

# Mit Loop: 3 Zeilen reichen!
for number_of_file in [1, 2, 3, 4, 5]:
    print(f"Datei my_document{number_of_file}.xls wird geladen")
Output
Datei my_document1.xls wird geladen Datei my_document2.xls wird geladen Datei my_document3.xls wird geladen Datei my_document4.xls wird geladen Datei my_document5.xls wird geladen

Der for-Loop wiederholt einen Codeblock für jedes Element in einer Sequenz.

PYTHON
groceries = ['Bananen', 'Äpfel', 'Zucker', 'Butter', 'Salat']

for item in groceries:
    print(f"Heute muss ich noch {item} einkaufen.")
Output
Heute muss ich noch Bananen einkaufen. Heute muss ich noch Äpfel einkaufen. Heute muss ich noch Zucker einkaufen. Heute muss ich noch Butter einkaufen. Heute muss ich noch Salat einkaufen.

Loop-Systematik: das allgemeine Muster

① Vor dem Loop
Variablen initialisieren:
leere Liste, Zähler = 0
② Im Loop
Iterationsvariable verarbeiten, Ergebnisse sammeln
③ Nach dem Loop
Ergebnisse nutzen: ausgeben, weiterverarbeiten
PYTHON
# ① Vor dem Loop: Initialisierungen
count = 0
total = 0
values = [9, 41, 12, 3, 74, 15]

# ② Im Loop: verarbeiten
for v in values:
    count += 1
    total += v

# ③ Nach dem Loop: Ergebnisse nutzen
print(f"Anzahl: {count}, Summe: {total}, Mittelwert: {total/count}")
Output
Anzahl: 6, Summe: 154, Mittelwert: 25.666666666666668
📌
Naming-Convention: Einzahl vs. Mehrzahl

Der Name der Iterationsvariablen sollte ein Element des Iterables repräsentieren:

for car in cars ✅ — car = ein Auto, cars = viele Autos
for item in groceries ✅ — item = ein Artikel
for x in cars ⚠️ — was ist x? unklar

💡
Iterationsvariable bleibt nach dem Loop verfügbar!

Nach dem Loop-Ende steht die Iterationsvariable auf dem letzten Wert und kann weiter verwendet werden:

for number in [1, 2, 3, 4, 5]:
    pass   # nichts tun
print(number)   # → 5   (letzter Wert!)
📐
Einrückung (Indentation) ist Pflicht!

Alles was zum Loop gehört, muss mit 4 Leerzeichen (oder 1 Tab) eingerückt sein. Python erkennt damit den "Körper" des Loops. Falsches Einrücken führt zu einem IndentationError.

Mehrere Zeilen im Loop-Body

PYTHON
# Der Loop-Body kann beliebig lang sein:
for number in [1, 2, 3, 4, 5]:
    squared = number ** 2
    print(f"Das Quadrat von {number} ist {squared}")
    print(f"Und wenn ich 1 abziehe, kommt {squared - 1} raus.")

print("Außerhalb des Loops!")  # ← wird nur 1× ausgeführt

Die range()-Funktion

range() erzeugt eine Zahlenfolge — perfekt für Loops mit Zähler.

PYTHON
range(5)          # → 0, 1, 2, 3, 4         (stop=5, start=0)
range(1, 6)       # → 1, 2, 3, 4, 5         (start=1, stop=6)
range(1, 10, 2)   # → 1, 3, 5, 7, 9         (start, stop, schritt)
range(6, 1, -1)   # → 6, 5, 4, 3, 2         (rückwärts)

# Praxis: Liste durchlaufen mit Index
my_garage = ['bmw', 'audi', 'toyota', 'skoda', 'vw']
for i in range(len(my_garage)):
    print(f"Auto {i+1}: {my_garage[i]}")
Output
Auto 1: bmw Auto 2: audi Auto 3: toyota Auto 4: skoda Auto 5: vw

Laufender Zähler im Loop

PYTHON
count = 0
groceries = ['Bananen', 'Äpfel', 'Zucker', 'Butter', 'Salat']

for item in groceries:
    count = count + 1     # Zähler hochzählen

print(f"Insgesamt {count} Lebensmittel.")
Output
Insgesamt 5 Lebensmittel.

# Zwei Listen mit gleichem Index zusammen verarbeiten

🧘 Coding Yoga: for-Loop I — Dictionary + String Aufgabe ▶

Was gibt print(sentence) aus?

PYTHON
my_dictionary = {
    0: ' ', 3: 'eat', 2: 'apple',
    4: 's', 1: 'Bob', 5: '.'
}
words = [1, 0, 3, 4, 0, 2, 4, 5]
sentence = ''
for w in words:
    sentence += my_dictionary[w]
print(sentence)
LÖSUNG
# w=1→'Bob', w=0→' ', w=3→'eat', w=4→'s'
# w=0→' ', w=2→'apple', w=4→'s', w=5→'.'
# Ergebnis: "Bob eats apples."
🧘 Coding Yoga: for-Loop II — Dictionary + range() Aufgabe ▶

Was gibt print(a_number) aus?

PYTHON
my_dict = {
    'dictionaries': ['are', 'funny'],
    'because': ['they', 'contain', 'lists']
}
a_number = len(my_dict)
for i in range(len(my_dict['dictionaries'])):
    a_number = a_number + len(my_dict['because'][i])
print(a_number)
LÖSUNG
# len(my_dict) = 2  → a_number = 2
# range(len(['are','funny'])) = range(2) = [0, 1]
# i=0: len(my_dict['because'][0]) = len('they') = 4
#      a_number = 2 + 4 = 6
# i=1: len(my_dict['because'][1]) = len('contain') = 7
#      a_number = 6 + 7 = 13
# Ergebnis: 13
🧘 Coding Yoga: for-Loop III — Caesar-Verschlüsselung Fortgeschritten ▶

Was gibt print(encrypted) aus?

PYTHON
alphabet = 'abcdefghijklmnopqrstuvwyz'
to_be_encrypted = 'hello world'
encrypted = ''

for letter in to_be_encrypted:
    new_index = alphabet.find(letter) - 1
    if new_index > 0:
        encrypted += alphabet[new_index]
    else:
        encrypted += letter

print(encrypted)
LÖSUNG
# Jeder Buchstabe wird durch den Buchstaben davor ersetzt:
# h → g, e → d, l → k, l → k, o → n
# ' ' → .find(' ') = -1, -1 > 0 ist False → Leerzeichen bleibt
# w → v, o → n, r → q, l → k, d → c
# Ergebnis: 'gdkkn vnqkc'

⏳ while-Loop

Der while-Loop läuft solange eine Bedingung True ist. Wenn man nicht weiß, wie viele Durchläufe nötig sind → while statt for.

PYTHON
n = 0
while n < 5:
    print(n)
    n = n + 1   # WICHTIG: Zähler erhöhen, sonst Endlosloop!
Output
0 1 2 3 4
⚠️
Vorsicht: Endlosloop!

while True: läuft für immer, falls kein break vorhanden ist. Im Jupyter Notebook: Kernel-Button (■) klicken zum Stoppen.

🧘 Coding Yoga: while-Loop I Aufgabe ▶

Was gibt print(a_number) am Ende aus?

PYTHON
my_list = [2, 1]
a_number = 3
while my_list != []:
    this_one = my_list.pop()
    a_number = a_number + this_one * len(my_list)
print(a_number)
LÖSUNG
# Durchlauf 1: pop()→1, my_list=[2], len=1
#   a_number = 3 + 1 * 1 = 4
# Durchlauf 2: pop()→2, my_list=[], len=0
#   a_number = 4 + 2 * 0 = 4
# Ergebnis: 4
🧘 Coding Yoga: while-Loop II — Bedingung + append() Aufgabe ▶

Was gibt print(sum(another_list)) aus?

PYTHON
i = 2
another_list = []
while i % 3:
    i = i + 2
    another_list.append(i * (len(another_list) + 1))
print(sum(another_list))
LÖSUNG
# Start: i=2, 2%3=2 (truthy) → Schleife läuft
# Durchlauf 1: i=4, len=0 → append(4*(0+1)) = append(4)
#   another_list=[4], 4%3=1 (truthy) → weiter
# Durchlauf 2: i=6, len=1 → append(6*(1+1)) = append(12)
#   another_list=[4,12], 6%3=0 (falsy) → STOP
# sum([4, 12]) = 16
# Ergebnis: 16

⚖️ for vs. while — wann was?

🔁 for-Loop
  • Iteriert über ein Iterable
  • Läuft für jedes Element genau einmal
  • Man weiß vorher, wie viele Durchläufe
  • for item in my_list
⏳ while-Loop
  • Läuft solange eine Bedingung True ist
  • Anzahl der Durchläufe variabel
  • Man weiß nicht vorher, wie oft
  • while condition == True
PYTHON
# for → wenn Iterable bekannt
for item in ['a', 'b', 'c']:
    print(item)

# while → wenn Abbruchbedingung unbekannt
import random
count = 0
while random.randint(1, 6) != 6:   # würfeln bis 6
    count += 1
print(f"Gebraucht: {count} Versuche")

🤔 if / elif / else

Mit if-Statements führt Python Code nur aus, wenn eine Bedingung erfüllt ist.

PYTHON
my_age = 20

if my_age < 4:
    print("Eintritt: 5 EUR")
elif my_age < 18:
    print("Eintritt: 10 EUR")
else:
    print("Eintritt: 15 EUR")
Output
Eintritt: 15 EUR
if
Erste Bedingung — wird immer geprüft
elif
Weitere Bedingung — nur wenn if False
else
Fallback — wenn alle anderen False

Verschachtelte Indentation (Ebenen)

Wenn man if innerhalb eines for-Loops nutzt, entstehen mehrere Einrück-Ebenen:

PYTHON
cars = ['audi', 'bmw', 'toyota', 'mercedes']

for car in cars:               # Indentation-Ebene 0
    print(car)                 # Indentation-Ebene 1 (im for)
    if car == 'bmw':           # Indentation-Ebene 1 (im for)
        print(car.upper())     # Indentation-Ebene 2 (im for+if)
    else:                      # Indentation-Ebene 1
        print(car.title())     # Indentation-Ebene 2

print("Alle Autos verarbeitet.")   # Ebene 0 (außerhalb for)
Output
audi Audi bmw BMW toyota Toyota mercedes Mercedes Alle Autos verarbeitet.

if mit in-Operator

PYTHON
# in-Operator prüft ob Element in einer Liste oder String ist
requested_toppings = ['Tomaten', 'Käse', 'Pilze']

if 'Tomaten' in requested_toppings:
    print("Tomate drauf!")

if 'Ananas' in requested_toppings:
    print("Ananas drauf!")
else:
    print("Keine Ananas.")
Output
Tomate drauf! Keine Ananas.

if zum Suchen in einer Liste

PYTHON
found = False
values = [9, 41, 12, 3, 74, 15]

for value in values:
    if value == 3:
        found = True
    print(f"{found}  →  Wert: {value}")

print(f"Gefunden: {found}")
Output
False → Wert: 9 False → Wert: 41 False → Wert: 12 True → Wert: 3 True → Wert: 74 True → Wert: 15 Gefunden: True

🔀 Mehrere if-Statements hintereinander

📌
Wichtiger Unterschied: if-elif-else vs. mehrere ifs

Bei if-elif-else stoppt Python nach dem ersten True und führt den Rest nicht mehr aus.
Bei mehreren getrennten if-Blöcken werden alle Bedingungen unabhängig voneinander geprüft!

if-elif-else: stoppt beim ersten True
PYTHON
x = 5
if x > 0:
    print("positiv")   # ← True!
elif x > 3:
    print(">3")        # ← wird NICHT geprüft
else:
    print("sonst")
Output
positiv
Mehrere ifs: ALLE werden geprüft
PYTHON
x = 5
if x > 0:
    print("positiv")   # ← ausgeführt
if x > 3:
    print(">3")        # ← AUCH ausgeführt
if x > 10:
    print(">10")       # ← nicht ausgeführt
Output
positiv >3

Praxisbeispiel: Pizza-Bestellung — alle Zutaten sollen unabhängig geprüft werden:

PYTHON
requested_toppings = ['Tomaten', 'Käse']

if 'Tomaten' in requested_toppings:
    print("Tomaten werden aufgelegt.")         # ← wird ausgeführt

if 'Salami' in requested_toppings:
    print("Salami wird aufgelegt.")            # ← wird AUCH geprüft (False → kein Print)

if 'Käse' in requested_toppings:
    print("Käse wird draufgestreut.")          # ← wird ausgeführt

print("Pizza ist fertig!")
Output
Tomaten werden aufgelegt. Käse wird draufgestreut. Pizza ist fertig!

Kettenvergleiche & Short-Circuit Evaluation

PYTHON
# Kettenvergleiche: Python erlaubt mathematische Schreibweise
a = 1 < 2 < 3 < 4 < 5   # → True  (prüft jedes Paar!)
b = 1 < 3 > 2            # → True

# Short-Circuit Evaluation: Python wertet von links nach rechts aus
# Bei 'or': wenn das erste True ist → zweites wird NICHT mehr ausgewertet
x = True
if x or undefined_variable:   # → kein NameError, weil x=True genügt
    print("OK")

# Bei 'and': wenn das erste False ist → zweites wird NICHT ausgewertet
if False and undefined_variable:   # → kein NameError!
    print("Nie")
🧘 Coding Yoga: if + for I Aufgabe ▶

Was gibt print(message) am Ende aus?

PYTHON
my_list = [1, 2, 3, 4, 5]
running_index = 0
for item in my_list:
    if running_index == 0:
        message = 'Are you serious'
    else:
        if my_list[running_index] % my_list[running_index - 1]:
            message += '!'
        else:
            message += '?'
    running_index += 1
print(message)
LÖSUNG
# Index 0: message = 'Are you serious'
# Index 1: 2 % 1 = 0 → Falsy → '?'  → 'Are you serious?'
# Index 2: 3 % 2 = 1 → Truthy → '!' → 'Are you serious?!'
# Index 3: 4 % 3 = 1 → Truthy → '!' → 'Are you serious?!!'
# Index 4: 5 % 4 = 1 → Truthy → '!' → 'Are you serious?!!!'
# Ergebnis: "Are you serious?!!!"
🧘 Coding Yoga: if + for II — komplexe Bedingungen Fortgeschritten ▶

Was wird bei jedem Schritt geprinted? Was ist der Endwert von mixed_stuff?

PYTHON
reverse_it = list(range(6, 1, -1))
mixed_stuff = 4
for number in reverse_it:
    if number % 2:
        mixed_stuff += 4
    elif number % 3:
        mixed_stuff -= 3
    elif number % 4:
        mixed_stuff /= 2
    else:
        mixed_stuff *= -1
    print(mixed_stuff)
LÖSUNG
# reverse_it = [6, 5, 4, 3, 2]
# number=6: 6%2=0, 6%3=0, 6%4=2→/2 → mixed=2.0 → print(2.0)
# number=5: 5%2=1→+4         → mixed=6.0 → print(6.0)
# number=4: 4%2=0, 4%3=1→-3  → mixed=3.0 → print(3.0)
# number=3: 3%2=1→+4         → mixed=7.0 → print(7.0)
# number=2: 2%2=0, 2%3=2→-3  → mixed=4.0 → print(4.0)
# Endwert: 4.0

⏸️ break & continue

break — Loop sofort beenden

PYTHON
while True:
    line = input('> ')
    if line == 'done':
        break          # beendet den Loop
    print(line)
print('Done!')

continue — Durchlauf überspringen

PYTHON
while True:
    line = input('> ')
    if line[0] == '#':
        continue       # Zeile ignorieren
    if line == 'done':
        break
    print(line)

reversed() — Liste rückwärts durchlaufen

PYTHON
# reversed() gibt einen Iterator in umgekehrter Reihenfolge zurück
my_list = [1, 2, 3, 4, 5]

for item in reversed(my_list):
    print(item)   # → 5, 4, 3, 2, 1

# Als Liste:
list(reversed(my_list))           # → [5, 4, 3, 2, 1]

# Kombiniert mit range:
list(reversed(list(range(6, 1, -1))))   # → [2, 3, 4, 5, 6]

⚡ List Comprehensions

Eine List Comprehension ist eine kompakte Schreibweise für einen for-Loop, der eine neue Liste erzeugt. Spart Platz und ist pythonischer.

Langer Weg
PYTHON
new_list = []
for x in range(10):
    if x % 2:
        new_list.append(x ** 2)
print(new_list)
⭐ Comprehension
PYTHON
new_list = [x ** 2
            for x in range(10)
            if x % 2]
print(new_list)
Output (beide identisch)
[1, 9, 25, 49, 81]

Syntax-Erklärung

PYTHON
#  [  ausdruck   for  variable  in  sequenz   if  bedingung  ]
   [  x ** 2    for  x         in  range(10)  if  x % 2     ]

# Mit if-else OHNE Filter (ternärer Ausdruck):
[x ** 2 if x % 2 else x * 2 for x in range(5)]
# → [0, 1, 4, 9, 8]

# String-Beispiel:
' '.join([x.lower() for x in ['I', 'AM', 'NOT', 'SHOUTING']])
# → 'i am not shouting'
